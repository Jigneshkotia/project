Ordered List

An ordered list typically is a numbered list of items. HTML 3.0 gives you the ability to control the sequence number - to continue where the previous list left off, or to start at a particular number.

Radio Button

In HTML, a radio button, also known as an option button, is an interactive element that allows a user to select exactly one option from a group of predefined choices. It is represented by a small circle that can be either empty (unselected) or filled in (selected). When a user clicks on a radio button, it becomes selected, and any other radio buttons in the same group are automatically deselected.

check boxes

In HTML, a checkbox is an interactive element that allows a user to select one or more options from a set of predefined choices. Unlike radio buttons, which limit users to selecting only one option from a group, checkboxes allow for multiple selections.

Headings

In HTML, headings are defined using the <h1> to <h6> tags. These tags are used to structure the content of a web page by dividing it into sections of different importance. The <h1> tag is used for the most important heading, while the <h6> tag is used for the least important heading.

Paragraph

In HTML, a paragraph is defined using the <p> tag. Paragraphs are fundamental building blocks of web content, used to group related text into logical units. They provide a clear separation between different ideas or sections of text, making the content easier to read and understand. 


                                                    <p><b>This text is bold</b></p>
                                                    <p><i>This text is italic</i></p>
                                                    <p>This is<sub> subscript</sub> and <sup>superscript</sup></p>

Formatting Tags

HTML formatting tags are used to modify the appearance and structure of text and other elements on a web page. They provide various ways to style and organize content, making it more visually appealing, easier to read, and better suited for different purposes.
